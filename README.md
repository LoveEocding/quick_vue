## Install

```bash
# Install vant 2.x for Vue 2 project
npm i quick_vue -S
```

## Quickstart

```js
import Vue from 'vue';
import { Wheel } from 'quick_vue';
import 'quick_vue/lib/index.css';

Vue.use(Wheel);
```

See more in [Quickstart](http://hbtest.wahaha.com.cn/quick_vue).
