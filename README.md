## Install

```bash
# Install vant 2.x for Vue 2 project
npm i quick_vue -S
```

## Quickstart

```js
import Vue from 'vue';
import { Tigger } from 'quick_vue';
import 'quick_vue/lib/index.css';

Vue.use(Tigger);
```

See more in [Quickstart]().
